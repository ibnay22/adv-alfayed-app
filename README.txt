# ⚖️ Adv. Alfayed — Practice Manager
## Complete Deployment Guide

---

## WHAT YOU NEED
- A computer (laptop or desktop) with internet
- 20 minutes
- This folder (AdvAlfayed_App)

---

## STEP 1 — Install Node.js on the Computer
1. Go to: https://nodejs.org
2. Click the big green "LTS" download button
3. Install it (just click Next → Next → Install)
4. Done. Node.js is now installed.

---

## STEP 2 — Create a FREE GitHub Account
1. Go to: https://github.com
2. Click "Sign up" (top right)
3. Enter your email, create a password, choose a username
   → Suggested username: adv-alfayed
4. Verify your email
5. You now have a GitHub account

---

## STEP 3 — Create a FREE Netlify Account
1. Go to: https://netlify.com
2. Click "Sign up"
3. Choose "Sign up with GitHub" → Click "Authorize"
4. You are now logged into Netlify

---

## STEP 4 — Upload to GitHub
1. Go to: https://github.com/new
2. Repository name: adv-alfayed-app
3. Keep it "Public"
4. Click "Create repository"
5. On the next screen, click "uploading an existing file"
6. Drag and drop EVERYTHING inside the AdvAlfayed_App folder
   (drag all files AND the src and public folders together)
7. Scroll down → Click "Commit changes"
8. Wait 10 seconds for upload to complete

---

## STEP 5 — Deploy on Netlify (Automatic!)
1. Go to: https://app.netlify.com
2. Click "Add new site" → "Import an existing project"
3. Click "GitHub"
4. Click "adv-alfayed-app" in the list
5. Build settings will auto-fill from netlify.toml:
   → Build command: npm install && npm run build
   → Publish directory: dist
6. Click "Deploy site"
7. Wait 2-3 minutes while it builds

You will get a link like:
   https://adv-alfayed-app.netlify.app

That is your permanent app link! Share this link with no one — it is private to you.

---

## STEP 6 — Install on Your Android Phone (30 seconds)
1. Open Chrome on your Android phone
2. Type your Netlify link in the address bar
3. Wait for the app to fully load
4. Tap the THREE DOTS (⋮) menu at top right of Chrome
5. Tap "Add to Home screen"
6. Name it: Adv. Alfayed
7. Tap "Add"
8. Tap "Add to Home screen" again if asked

✅ The Adv. Alfayed icon now appears on your phone home screen.
   Tap it — it opens like a real app, full screen, no browser bar!
